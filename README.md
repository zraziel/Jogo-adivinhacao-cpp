# 🎯 Jogo de Adivinhação em C++

Projeto desenvolvido em **C++** durante meus estudos no curso de **Análise e Desenvolvimento de Sistemas.**  
O objetivo foi praticar **lógica de programação**, **funções**, **condições** e **manipulação de arquivos**.

---

## 🕹️ Sobre o jogo

O programa sorteia um número secreto, e o jogador precisa adivinhar.  
O jogo informa se o número digitado é **maior ou menor** que o correto e calcula uma **pontuação final** com base nas tentativas.

Níveis de dificuldade:
- 🟢 **Fácil:** 0 a 100  
- 🟡 **Médio:** 0 a 300  
- 🔴 **Difícil:** 0 a 500  

---

## 💾 Funcionalidades

- Escolha de nível de dificuldade  
- Sistema de pontuação  
- Armazenamento de recordes em arquivo  
- Feedback ao jogador a cada tentativa  
- Interface simples via terminal  

---

## ⚙️ Tecnologias utilizadas

- Linguagem **C++**  
- Biblioteca `<fstream>`  
- Estruturas de repetição e condicionais  
- Geração de números aleatórios  
- Modularização com funções
